- Governed access/classification/retention compliance is 37.0%; this is the strongest remediation workstream because it controls whether content is appropriate for AI retrieval.

- The ranked queue contains 767 non-resolved remediation items, including 323 P1 governance gaps.

Recommended sequence: resolve P1 governance exceptions first, refresh stale high-criticality sources second, then complete metadata, lineage, and semantic labels for assets already included in retrieval.
